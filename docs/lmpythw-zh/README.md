## 下载

### Docker

```
docker pull apachecn0/lmpythw-zh
docker run -tid -p <port>:80 apachecn0/lmpythw-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install lmpythw-zh
lmpythw-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g lmpythw-zh
lmpythw-zh <port>
# 访问 http://localhost:{port} 查看文档
```
