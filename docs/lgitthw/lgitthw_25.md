## 25.2 切换到 master 分支

要在分支间切换，只需使用 `git checkout` 命令。

```
$ git checkout master
$ cat lib/hello.rb
```

```
$ git checkout master
Switched to branch 'master'
$ cat lib/hello.rb
# Default is World
# Author: Jim Weirich (jim@somewhere.com)
name = ARGV.first || "World"

puts "Hello, #{name}!"
```

你现在在 master 分支上。你可以判断 `hello.rb` 文件是否使用 `Greeter` 类。

## 25.3 回到 Greet 分支

```
$ git checkout greet
$ cat lib/hello.rb
```

```
$ git checkout greet
Switched to branch 'greet'
$ cat lib/hello.rb
require 'greeter'

# Default is World
name = ARGV.first || "World"

greeter = Greeter.new(name)
puts greeter.greet
```

`lib/hello.rb` 的内容确定我们回到了 greet 分支上。
