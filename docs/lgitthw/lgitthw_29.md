# 第二十九章 创建冲突

## 29.1 目的

在 master 分支中创建一个冲突更改。

## 29.2 切换到 master 并创建冲突

切换到 master 分支并做这个更改：

```
$ git checkout master
```

文件：lib/hello.rb

```
puts "What's your name"
my_name = gets.strip

puts "Hello, #{my_name}!"
```

```
$ git add lib/hello.rb
$ git commit -m "Made interactive"
```
