# 笨办法学 Git

来源：[`learngit.selfhostedserver.com/`](https://learngit.selfhostedserver.com/)

*Jim Weirich 著 / 徐小东 译*

## 内容简介

本书强调通过实践来掌握 Git 的基本用法，其中包含 51 个动手实验。这些实验经过精心设计，篇幅皆十分短小，只需数分钟时间便可完成。对于想要快速学习 Git 的朋友而言，这是一份难得的指南。
