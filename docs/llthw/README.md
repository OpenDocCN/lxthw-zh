## 下载

### Docker

```
docker pull apachecn0/llthw-zh
docker run -tid -p <port>:80 apachecn0/llthw-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g llthw-zh
llthw-zh <port>
# 访问 http://localhost:{port} 查看文档
```
