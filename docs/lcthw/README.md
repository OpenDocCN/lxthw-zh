## 下载

### Docker

```
docker pull apachecn0/lcthw-zh
docker run -tid -p <port>:80 apachecn0/lcthw-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g lcthw-zh
lcthw-zh <port>
# 访问 http://localhost:{port} 查看文档
```
